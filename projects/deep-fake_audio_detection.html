This project aims to develop a high-performance system to detect AI-generated synthetic speech using deep learning techniques. The detection pipeline includes fine-tuned state-of-the-art models like AASIST (attention-based graph network), RawNet2 (end-to-end raw waveform model), and LCNN (lightweight CNN using Max-Feature-Map).

We used the ASVspoof2019 dataset for training and validation and performed fine-tuning with cosine learning rate scheduling and custom dataloaders for optimal performance in low-data scenarios. Each model was evaluated on its precision and robustness in differentiating between real and spoofed audio.

Extensive documentation was created with detailed experimental results, architectural comparisons, and research-backed justifications to ensure transparency and reproducibility. The final model setup is deployable and highly effective in real-world applications for detecting deepfake audio threats.
