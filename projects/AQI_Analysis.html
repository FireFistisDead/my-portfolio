Hacktober AQI Analysis

October 2024 GitHub Repository →

This open-source project was developed as part of Hacktoberfest 2024 and focused on analyzing Air Quality Index (AQI) data across various cities and timeframes. The goal was to derive meaningful insights from real-time air quality metrics to help raise awareness and support research on pollution trends.

Using Python and libraries like Pandas, the project processed large-scale AQI datasets to compute air quality trends, average pollutant levels, and historical comparisons. Data visualization tools like Matplotlib and Plotly were employed to create interactive dashboards and graphs for easy interpretation of results.

We also explored machine learning techniques to enhance AQI prediction capabilities. Regression models were trained on historical data to forecast future pollution levels, helping users and policymakers make informed decisions.
